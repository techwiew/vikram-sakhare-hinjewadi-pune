# Vikram Sakhare Website (React)

React + Vite website for `Vikram Sakhare (Sarpanch, Hinjewadi Grampanchayat)`.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Update Content

Edit `public/data.json`.

## Security Baseline Included

- No `dangerouslySetInnerHTML`
- Safe external URL validation before link render
- CSP and security meta tags in `index.html`
- Hostinger/Apache security headers in `public/.htaccess`
- `rel="noopener noreferrer"` for external links
